Locket Widget Application

September 2024 – October 2024

A web-based application inspired by widget-style social interaction, allowing users to share and manage photos and messages in a lightweight interface.

🚀 Project Overview

This project focuses on designing and developing a full-stack web application, including system architecture planning, database design, and frontend implementation.

🛠️ Key Contributions
Designed and analyzed system architecture using:
Use-case diagrams
Entity-Relationship (ER) diagrams
Relational database models
Implemented database operations using Oracle SQL, including:
User management
Posts and photo sharing
Messaging system
Photo album management
Developed and styled the user interface using:
HTML5
CSS3
🎯 Skills Demonstrated
Database Design & SQL (Oracle)
System Analysis & Design
Frontend Development
Data Modeling (ER Diagram, Relational Model)
